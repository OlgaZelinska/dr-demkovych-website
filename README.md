# Dr.Demkovych - Професійна хірургічна допомога

Сайт хірурга Демковича Олександра з більш ніж 15-річним досвідом роботи.

## 🧾 Про сайт

Це сучасний адаптивний сайт на React + TypeScript + Vite + Tailwind CSS для хірурга-професіонала.

## 🛠️ Стек технологій

- **React 18** - бібліотека для інтерфейсів
- **TypeScript** - типізований JavaScript
- **Vite** - швидка збірка проекту
- **Tailwind CSS** - стилізація
- **React Router** - навігація між сторінками
- **React Query** - керування API запитами

## 📁 Структура проекту

```
src/
├── assets/          # Зображення та ресурси
├── components/      # React компоненти
│   ├── layout/      # Header, Footer
│   ├── sections/    # Секції сторінок
│   └── ui/          # UI елементи
├── lib/             # Бібліотеки та функції
├── pages/           # Сторінки
└── routes/          # Маршрути
```

## 🚀 Як запустити

```bash
# Встановлення залежностей
npm install

# Розробка (з перезавантаженням)
npm run dev

# Збірка для продакшн
npm run build

# Попередній перегляд
npm run preview
```
